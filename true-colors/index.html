True Colors is a visual experiment that evaluates the graphic symbolism of United States flags. Typically, the design of these flags is arbitrary, or is symbolic but does not accurately represent the state in a non-objective manner.

Through utilizing data from the 2015 American Community Survey, True Colors generates flag designs based on state demographics, percentages, and rankings, allowing citizens to identify certain aspects of the state at a glance. Each flag is uniquely representative of the state it stands for, and may be updated as these numbers change.
